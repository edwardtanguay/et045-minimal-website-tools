Alpine.js and Pico.css make form building much more straight-forward and provides many simplifications that improve the user-experience and standardize the data being sent to the server, e.g. checkboxes are sent as boolean true or false instead of a name or blank as on standard HTML forms.

The x-model attribute works well with forms and provides a state-variable experience similar to React and Vue.js where the application is aware of the changes of each of the form controls as the user types and clicks on these controls, which is shown below on the right.

Also note that we use the Pico.CSS element <div class="grid"> which gives this grid a nice responsive feature, e.g. the right side is shown below the form on mobile view.
