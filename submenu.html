Intro

With simple HTML attributes, Alpine.js allows you to turn on and off areas on the screen, or add or remove classes based on the value of a variable.

Similarities to Vue.js

Alpine.js has a syntax that reminds one of Vue.js, but the library is much simpler. For instance, you don't have to add a JavaScript object on your page in order to use it. You simply add variables in HTML elements themselves.

					<body x-data="{subpage: 'p1'}">

					<li @click="subpage = 'p1'" :class="subpage==='p1' ?
					'active' : ''">Intro</li>

					<section x-show="subpage==='p1'">

Good user experience on mobile devices

Because this form of menus do not require a round-trip to the server, the user experiences the site as much faster on mobile devices.

Bookmarking must be programmed

Note that with this solution, when you click a menu link, it is not reflected in the URL.

This would have to be programmed with JavaScript, similar to the Router in this Vite Vanilla starter.

Other uses

Alpine's feature that allows you to easily define state variables which can turn hide and show various elements on the page, or change their appearance based on what the use sets with button and other controls, can be used in many ways:

  • interactive forms
  • games
  • learning tools such as flashcards
  • creative layouts and menus that allow the user to turn on only what he wants to see
